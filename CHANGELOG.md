# Changelog

All notable changes to moneyfrisk are documented here.

## [0.1.0] — 2026-06-30

First release.

### Added
- The skill (`skill/SKILL.md`) — frisks the money lines in your diff for floating-point currency bugs.
- One-line installers (`install.sh`, `install.ps1`); multi-harness.
- Docs (`docs/`), real runs (`CASES.md`), i18n (EN / ID / zh-CN), key-free demo.
- MIT license, CONTRIBUTING, commitlint config + CI + opt-in hook.
